# 📊 HR Analytics Dashboard

## 📌 Overview

This project analyzes employee data to understand attrition trends and workforce distribution using Power BI.

## 🔍 Key Insights

* Attrition Rate: 16.12%
* Highest attrition in Sales department
* Majority employees in 25–34 age group
* Job satisfaction varies across roles

## 🛠 Tools Used

* Power BI
* Excel
* DAX
